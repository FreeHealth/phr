Resupply Date

Enter the date a new supply of medication is needed using the format YYYY/MM/DD.

If you choose to enter a resupply date, an exact date is required.

The resupply date should be set to the date you need to purchase, refill, or obtain a new prescription for the medication.

The Date Reminders link at the top of the PHR page will display a list of medication resupply due dates along with other reminders.
