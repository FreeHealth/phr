About the PHR Account ID

Your PHR account ID is the unique identifier you will use to access your Personal Health Records.

Longer and more complex account IDs are more secure; however, you should choose an account ID that will be easy for you to remember. Please avoid using any personal identifiers in this field to ensure your privacy.

Your PHR account ID

  • can be from 6 to 32 characters long
  • must begin with a letter
  • can contain a combination of letters, numbers, and underscore (_) characters
  • is not case-sensitive

You will be able to create multiple Personal Health Records using the same PHR account ID.
