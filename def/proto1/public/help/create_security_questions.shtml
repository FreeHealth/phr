Security Questions

To ensure the security of your PHR account, we ask you to set up 3 security questions and answers which will allow you to verify your identity should you forget your password. You will be able to automatically recover or reset your account ID or password by verifying your identity by supplying the answers to these questions. Be sure to select questions that have answers you will remember over time.

Please select and answer 2 security questions from our list and then create 1 of your own security questions and answers.
