Include Test Data from Other Panels

Test measurements can be added individually and can then be displayed as a result in more than one test panel. The PHR system'a default is to display the test as you originally entered it – either by itself or as part of the panel you chose.

For example, you can record a potassium test result by itself by selecting "Potassium" from the list of tests, or you can enter it using the "Basic metabolic panel" (“chem 7”) panel or even as part of the "Comprehensive metabolic panel" (“chem12”). If you enter one potassium result by itself and another as part of the basic metabolic panel, the system will display the two values separately as shown below:

If you select the "Include test data from other panels" checkbox, the test will be displayed in all of the panels that contain a potassium result field.

Note: You will not be able to edit the test result that was entered individually or as part of another panel from within the related panel (e.g., in the example above, you will not be able to edit the single potassium value from within the basic metabolic panel).
