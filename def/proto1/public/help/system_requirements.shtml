What Are the System Requirements for Using the PHR?

Operating Systems

The PHR will work with the following operating systems:

  • Windows 8, 7, Vista, or XP
  • Macintosh OS X
  • Linux

Browsers

The PHR will work with the following browsers:

  • Mozilla Firefox Version 18.0 or later
  • Internet Explorer Version 9.0 or later
  • Google Chrome Version 24.0 or later
  • Safari Version 5.0 or later (not extensively tested, but we believe it works with all of the PHR's features)

Note: When accessing the PHR, users should avoid opening more than one instance of the PHR at the same time. Using multiple windows to view and enter data within the PHR may result in unexpected behavior such as errors when attempting to save data. This applies to both the basic and standard versions of the PHR.

Browser Settings

  • Enable JavaScript: Regardless of your browser type, you must have a browser that supports JavaScript and the browser must be set to enable JavaScript. (Does not apply to Basic HTML Mode.)
  • Enable Session Cookies: You must also have session cookies enabled in your browser to use the PHR.

Accessibility

  • Increasing and decreasing the size of the PHR text.
  • If you plan to use the PHR with assistive technology such as screen readers, please access the PHR using Basic HTML Mode.
