PHR Record Name

Create and enter a PHR Record Name for the person whose data is to be entered into this Personal Health Record. PHR record names may include:

  • From 1 to 255 characters
  • Spaces, numerals, and any special character except for "<"

Avoid using personal identifiers as record names. You may wish to use an alias or invented name such as "Self", "Son 1", or "Father" to ensure your privacy.

A record name can be changed by clicking the Demographics button underneath the user's name on the PHR Home page.

To edit an existing PHR, choose the a PHR record from the list PHRs on the PHR Home page.
