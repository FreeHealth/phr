Edit the Demographics (Name, Birth Date, Gender, or Race) of a PHR Record

You can change the following demographic characteristics of a PHR record:

  1. PHR Record Name
  2. Date of Birth
  3. Gender
  4. Race or Ethnicity

To edit the basic demographic information in a PHR record:

  1. On the PHR Home page, click the link to the name of the PHR you wish to change. This will expand the PHR entry to display action buttons for the individual's PHR.
  2. Select the Demographics button beneath the PHR name you wish to change.
  3. In the pop-up window, edit one or more of the four demographic fields of the PHR record.
  4. Click the Save button at the bottom of the form.
