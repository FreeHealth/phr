A title should be enclosed in the H1 tags. You can have only one title.

Under the title, paragraphs are enclosed in the P tags. There could have no paragraphs at all, or have more than one paragraph, each enclosed in its own P tags.

A list could also be in this place. See below to see how to create a list.

A group header should be enclosed in the H2 tags or in the H3 tags.

Under the group header you can have paragraphs, lists, tables and images.

There are two types of lists: ordered lists and unordered lists. Here are the examples.

Ordered List:

  1. The list is enclosed in the OL tags.
  2. Each list item is enclosed in the LI tags.
  3. Each list item starts with a sequential number.

Unordered List:

  • The list is enclosed in the UL tags.
  • Each list item is enclosed in the LI tags.
  • Each list item starts with a bullet (a small circle).

Links

Note:

A list of notes:

  • No other H tags (H4, H5 and etc.) should be used in the help files.
  • Usually no css settings are allowed in the help files.
  • All text should be within tags.
  • No JavaScript code is allowed in the help files.
  • B tags could be used anywhere within the text as appropriate.
  • Other tags such IMG may be added when it does not break the layout and the style of the help files.
  • Sometimes, you may need to set some attributes of a table, such as colspan, to control its format.
