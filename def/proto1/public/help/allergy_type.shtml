Allergy Type

Indicate the general class of allergy you have. The PHR classifies substances that cause allergic or dangerous reactions as:

  1. Food (e.g., nuts, shellfish)
  2. Medication classes (e.g., sulfa drugs, penicillin and derivatives)
  3. Medications specific (e.g., Amoxicillin, Codeine)
  4. Other (e.g., latex, ragweed)

Select one of these allergy types from the drop-down list.
