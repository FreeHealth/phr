View All Tests and Procedure Results

The Flowsheet page allows you to display a summary of all of your test and procedure results. You and your health care provider can quickly and easily track test results such as blood pressure, cholesterol, weight, and others over time.

To view results from your saved tests and procedures:

  1. Click the Flowsheet link on the Tests & Procedures page.
  2. Click the checkbox next to one or more of the listed tests or procedures. If you wish to see data for multiple test results, select the checkbox next to each set of results you wish to display.
  3. If desired, select any of the optional display settings.
  4. Click the Show Results Timeline button to display the selected test results.
