Group Data Columns

You may optionally group test results by week, month, or year which will allow you to view only the last value from a specific time period.

For example, if you have entered your weight every day for the last year but want to see the trend over the year, you could select either the By Week or By Month option so that you will only see the last value recorded in each week or month, respectively.

Note: You will not be able to edit results on the results timeline page if you group test results by week, month, or year. You must select None (which is the default option) to enable editing of results on the results timeline page.
