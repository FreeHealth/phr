Previously Saved Tests & Trackers

This section of the PHR contains all of the tests and trackers you have previously entered using the Add Test Results & Trackers page of the PHR.

What do I need to know about previously saved tests and trackers?

  • Nearly all medical test results and measures can be recorded in your PHR using the Add Test Results and Trackers link at the top of the Health Record Details page.
  • Once a test or tracker has been added to a PHR using the Add Tests Results & Trackers link, the most recent recorded result for that test or tracker will be displayed in the Previously Saved Tests & Trackers section of the Health Record Details page.
  • Click the View link next to the test or tracker name to display all of the previously entered values for the test.
  • Click the Add More link next to the test or tracker name to add another result for the same test.

What kind of information should I enter?

All fields highlighted with are required fields.

Date DoneEnter the date the observation, test, or specimen was collected.
Time DoneEnter the time of day the test was done.
Where DoneEnter the location where the test was performed. You may select from the list of general names such as Home, Hospital, Doctorâ€™s Office, etc. or enter your own location names.
CommentSummarize the results of the test. If the laboratory provides an overall summary of the meaning of the test results taken together, you may also enter that information here.
Next DueEnter the date the next test is due.
NameDisplays the names of the measurements taken as part of this test panel.
ValueThe value of the test or result. Some test results have numeric values, some are either positive or negative, and others may have free text as their result.
PreviousDisplays the value and date of the most recent test recorded prior to this entry session.
UnitsIf the field contains a drop-down list of units of measurement, select the units reported in the test results from the list. Begin typing the unit abbreviation to see a list of commonly used measurement units.You may also enter your own units of measure in this field.
RangeEnter the reference range for the test. Reference ranges define the values that are either statistically abnormal or not healthy. Test results within the range are considered normal.
