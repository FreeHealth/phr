Medication Status

Indicate whether or not you are currently taking the medication you have entered. If you are currently taking it, select the Active status from the drop-down menu.

If you have entered a medication that you have taken in the past, and are no longer taking, select the Stopped status.

By default, the PHR system displays a list of all medications labeled as Active. Click the Include stopped medications check box to display the complete list of all Active and Stopped medications.
