Download a PHR to a File

The PHR system allows you to download a PHR to your computer into either a Microsoft Excel or a Comma Separated Values (CSV) file format.

To download a PHR to a file on your computer

  1. Select the Download button under the name of the PHR you wish to download from the PHR Home page.
  2. The system will display a form that will allow you to choose either the Excel or CSV file format.
  3. Click the Go button.
