Download a PHR to a File

The PHR system allows you to download a PHR to your computer into either a Microsoft Excel or a Comma Separated Values (CSV) file format.

To download a PHR to a file on your computer

  1. On the PHR Management page, click the Export link next to the name of the PHR you wish to export.
  2. The system will display a page that will allow you to choose either the Excel or CSV file format.
  3. Click the Export button.
