Accessibility

The PHR application is heavily dependent on JavaScript, graphics, and CSS for its user interface. If you have difficulty using it, we recommend you try the Basic HTML Mode. The Basic mode provides nearly the same functionality but with a simpler (and plainer) user interface.

Note: When accessing the PHR, users should avoid opening more than one instance of the PHR at the same time. Using multiple windows to view and enter data within the PHR may result in unexpected behavior such as errors when attempting to save data. This applies to both the basic and standard versions of the PHR.
