Allergy Name

Enter the name of the substance that causes the allergy or dangerous reaction. For medicines, use the medication class rather than the brand name. For instance, if you are allergic to penicillin V, you are allergic to all penicillins, so you should enter Penicillin and Derivatives as the name.

If you do not find the specific substance you are allergic to in the master list, simply type the name as you know it.
