Adding New Tests Results and Health Trackers

To add new test results or health trackers or add a new result to a test or tracker:

  1. Click the Tests & Measures link on the PHR Record page.
  2. On the Tests and Procedures page, click the New link.
    • Either begin typing the name of the medical test or health tracker you want to add in the search box and then click the Search button, or
    • Click the list of panels link to browse the list of panels.
