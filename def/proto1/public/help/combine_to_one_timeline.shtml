Combine Records in One Timeline Table

Note:To edit data on the View & Edit Results Timeline page, be sure to uncheck this option. You will not be able to edit data while the Combine records in one timeline table option is selected.

The default display shows the results for each test result or health tracker in its own timeline from newest to oldest.

For example, if you entered a weight today and last week and a cholesterol two weeks ago and last year, today's weight would appear in the same column as the cholesterol from two weeks ago, and last week's weight would appear in the same column as last year's cholesterol:

If you choose the Combine records in one timeline table option, a single timeline for all of the test results will be displayed so that you can see when one test was done relative to another one.

For example, if you entered a weight today and last week and a cholesterol two weeks ago and last year, both of the weight measurements, which are the newest results, will appear on the timeline first, followed by the cholesterol result:
